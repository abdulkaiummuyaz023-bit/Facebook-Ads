## 📋 Campaign Management Process

### Facebook Ads Workflow

1. **Strategy & Audience Research** — Define campaign objective, target audience, and budget
2. **Campaign Creation** — Set up campaign, ad sets, and ad creatives in Meta Ads Manager
3. **Audience Setup** — Configure custom audiences, lookalikes, or interest-based targeting
4. **Ad Creative Production** — Video ads, image ads, carousel formats
5. **Launch & Monitor** — Track delivery, results, CPR, impressions, and reach
6. **Optimization** — Scale winning ad sets, pause low performers, A/B test creatives
7. **Reporting** — Performance and engagement column reporting with breakdown analysis

### Google Ads Workflow

1. **Account & Campaign Structure** — Set up account, campaigns, and ad groups
2. **Keyword Research** — Target relevant search terms for Search campaigns
3. **PMax Setup** — Asset groups, product feeds, audience signals for Performance Max
4. **Conversion Tracking** — Link GA4 and Google Ads for conversion data
5. **Bid Strategy** — Maximize conversions / Target ROAS configuration
6. **Performance Review** — Monitor ROAS, CPC, CTR, impressions, and conversion volume
7. **Campaign Optimization** — Budget reallocation, audience refinement, asset testing
